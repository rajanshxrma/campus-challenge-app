# Campus Challenge

Flutter app that helps students stay on track by completing daily challenges and seeing their progress.

## Team
- Team U30
- CSC 4360 — Mobile App Dev
- Rajan Sharma, Michael Irving

## Features
- create, edit, delete challenges
- track daily completions
- streak counter and history
- dashboard with todays progress
- dark mode toggle
- weekly bar charts (fl_chart)
- export data to JSON

## Tech
- Flutter + Dart
- SQLite (sqflite) for local storage
- Provider for state management
- SharedPreferences for settings
- fl_chart for charts

## How to run
```
flutter pub get
flutter run
```
